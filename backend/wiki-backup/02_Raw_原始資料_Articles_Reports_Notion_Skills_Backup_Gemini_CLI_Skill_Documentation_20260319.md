# Gemini CLI 技能文檔 (Skill Documentation)
本文件詳述了 Gemini CLI 中目前可用的所有技能，按領域分類並以表格形式呈現。
## 啟動方式說明
所有技能均可透過 `/activate_skill <技能名稱>` 啟動。某些技能在特定情境下會由 AI 自動偵測並建議啟用。

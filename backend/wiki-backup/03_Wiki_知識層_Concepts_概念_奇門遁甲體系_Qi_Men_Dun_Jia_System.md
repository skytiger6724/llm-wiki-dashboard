# 核心概念：奇門遁甲體系 (Qi Men Dun Jia System)

## 核心定義
奇門遁甲是中國古代最高階的「時空博弈模型」。它透過將時間（干支）、空間（九宮）、環境（八卦）與神祕能量（九星、八門、八神）進行多維矩陣運算，來預測事態發展並尋求最優路徑。

## 體系 Schema
奇門遁甲本質上是一個**動態矩陣系統**：
- **靜態底層**：地盤九宮格（座標系）。
- **動態參數**：
    - **天盤 (九星)**：天時因素，宏觀趨勢。
    - **地盤 (奇儀)**：空間因素，物質基礎。
    - **人盤 (八門)**：人事因素，行動路徑。
    - **神盤 (八神)**：不可控因素，隨機擾動。

## 映射至現代科學
- **決策科學**：奇門遁甲 = 具備時空權重的多目標優化演算法。
- **電腦科學**：排盤 = 數據輸入與標準化；斷卦 = 規則匹配與推理引擎。

# **第二部分:💡有效的風格與技術指令（Prompt 技巧）**

### **️風格（Artistic Style）技巧**
|   |   |   |
|---|---|---|
|**類別**|**指令範例**|**說明**|
|攝影風格|寫實風格、電影燈光、微距攝影、低光、HDR、高級時尚風格|用於創造寫實和專業的攝影效果。|
|藝術流派|印象派、超現實主義、抽象藝術、波普藝術、巴洛克風格|創造經典藝術流派的影響。|
|數位藝術|數位繪畫、概念藝術、動漫風格、3D 渲染、虛幻引擎|用於電玩遊戲或數位插畫風格的視覺效果。|
### **📸技術細節技巧**
|   |   |   |
|---|---|---|
|**細節**|**指令範例**|**對圖像的影響**|
|光線|體積光、黃金時刻、工作室燈光、邊緣光（邊緣光）|決定光線的方向、柔和度和色彩。|

# **第三部分:️編輯現有照片的指令**
您可以使用 Gemini 不僅從零開始生成圖像,還能編輯您上傳的現有照片。
以下是一些可與上傳圖像一起使用的編輯指令:
- **色彩與色調:**「讓這張照片更鮮豔並增加對比度。」
- **背景變更:**「將背景替換為生動的熱帶海灘。不要改變主體本身。」
- **應用風格:**「將這張肖像轉換為安迪·沃荷的普普藝術風格。」

# **第四部分:🚫應避免的指令錯誤**
無法獲得理想結果的最常見原因包括:
|   |   |   |
|---|---|---|
|**錯誤**|**說明**|**如何修正?**|
|過於模糊|使用「製作一張漂亮的狗狗圖片」等籠統表述。|詳細描述主體和動作:「在森林中,金色光線照射下,笑容滿面的黃金獵犬。」|
|堆疊過多細節|在一個長句子中塞入 15 個不同的指示。|用逗號或破折號分隔指令來結構化,只聚焦於最重要的 5-6 個細節。|
|使用否定指令|添加否定詞如「不是紅色的汽車」。|AI 會聚焦於您想要什麼。直接說明您要的內容:「藍色汽車」。|

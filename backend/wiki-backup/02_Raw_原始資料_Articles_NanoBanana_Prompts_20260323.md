## 視覺美學提示詞對照表

| 場景 | 效果 | Prompt |
| :--- | :--- | :--- |
| **超寫實大合照** | 處理多位名人臉孔與特定光影，追求極致皮膚質感與細節。 | `Create a hyper-realistic, ultra-sharp, full-color large-format image featuring a massive group of celebrities... impeccably lifelike skin texture, pores, reflections...` |
| **2000 年代鏡像自拍** | 結構化 JSON，精準復刻千禧年初數位相機閃光燈美學。 | `{ "subject": { "description": "A young woman taking a mirror selfie...", "photography": { "camera_style": "early-2000s digital camera aesthetic", "lighting": "harsh super-flash" } } }` |
| **維密風格時尚大片** | 高端華麗、後台風格，強調珠寶閃耀與奢華感官。 | `Create a glamorous photoshoot in the style of Victoria's Secret. Lit by a camera flash to emphasize the shine of beads and crystals...` |
| **90 年代復古肖像** | 模擬底片相機直打閃光燈，營造隨性溫馨的個人空間氛圍。 | `Portrait of a beautiful young woman... captured with a 1990s-style camera using a direct front flash. 35mm lens creates a nostalgic glow.` |
| **矽谷商務形象照** | 隨性照片轉專業商務大頭照，指定 Sony A7III 搭配 85mm 鏡頭壓縮感。 | `Keep facial features exactly consistent. Professional navy blue business suit. Shot on a Sony A7III with an 85mm f/1.4 lens.` |
| **情感電影感攝影** | Kodak Portra 400 底片懷舊光影，適合黃金時段街頭氛圍。 | `Cinematic, emotional portrait shot on Kodak Portra 400 film. Urban street coffee shop window at Golden Hour.` |
| **專業頭像生成器** | 高解析度、充滿自信的工作室形象照，具備柔美散景效果。 | `Captured on an 85mm f/1.8 lens with a shallow depth of field, exquisite focus on the eyes, and beautiful soft bokeh.` |
| **聚光燈動漫肖像** | 結合超寫實與動漫風格，利用窄光束聚光燈營造戲劇張力。 | `Generate a hyperrealistic realistic-anime portrait standing in a completely black background. narrow beam spotlight on face.` |
| **浴室鏡面自拍** | 極具生活感的隨性自拍，強調純真臉孔與服裝反差。 | `{ "subject": { "description": "Young woman taking bathroom mirror selfie", "vibe": { "energy": "innocent face + sinful body = the whole game" } } }` |
| **黑板動漫藝術記錄** | 寫實記錄黑板上的粉筆畫，捕捉粉塵感與短暫存在的藝術美。 | `{ "intent": "Photorealistic documentation of a chalkboard art piece", "subject": { "primary_subject": "Intricate chalk drawing of Boa Hancock" } }` |
| **雪地小狗肖像** | 溫馨自然的冬季戶外合照，保持面部特徵與參考圖一致。 | `{ "image_description": { "subject": { "girl": { "expression": "puckering lips" }, "puppy": { "type": "small white puppy" } } } }` |
| **魚眼電影角色自拍** | 模擬 35mm 魚眼鏡頭強烈桶形畸變，拍出與角色的瘋狂自拍。 | `A hyper-realistic fisheye wide-angle selfie, vintage 35mm fisheye lens, heavy barrel distortion, wild exaggerated faces.` |
| **電影角色合照** | 保留 100% 面部特徵，瞬間穿越到電影拍攝現場。 | `Taking a selfie with [movie character] on the set of [movie name]. Keep the person exactly as shown in the reference image.` |
| **美術館展覽自拍** | 人物站在高端美術館，背景為質感厚重的古典油畫。 | `Commercial grade photograph posing inside a high-end museum exhibition. Behind them hangs a large, ornate framed classical oil painting.` |
| **數位相機螢幕顯示** | 模擬早期數位相機螢幕顯示效果，包含相機按鈕與螢幕顆粒。 | `A close-up shot displayed on the screen of a compact Canon digital camera. Camera body with buttons and dials visible.` |
| **雜誌封面肖像** | 製作帶有 "Nano Banana Pro" 標題的亮面雜誌封面，時尚感十足。 | `A photo of a glossy magazine cover, large bold words "Nano Banana Pro", dynamic portrait in green and banana yellow high-end fashion.` |
| **奢華產品攝影** | 漂浮在暗色水面上的高級產品照，利用倒影與漣漪營造空靈感。 | `Luxury product shot floating on dark water with [flower type] arranged around it. Reflections and ripples across the water.` |

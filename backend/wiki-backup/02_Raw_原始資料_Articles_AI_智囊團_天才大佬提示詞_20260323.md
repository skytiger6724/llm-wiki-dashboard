# AI 智囊團：7 位天才大佬提示詞手冊

**生成時間：** 2026年3月23日
**來源：** 阿七的 AI 踐行之路 (2026-03-20)
**核心主題：** 管理、決策、創新、設計、財富認知

---

## 7 位世界天才 AI 智囊團對照表

| 場景 | 效果 | Prompt (示例) |
| :--- | :--- | :--- |
| **1. 德魯克 (管理)**<br>遭遇職場瓶頸、個人效率低、團隊內耗 | 審視「有效產出」而非「忙碌程度」，從目標出發精準找到貢獻點。 | 我是 [職位]，正在處理 [任務]。請以德魯克的身份幫我分析：我的貢獻在哪裡？我應該放棄哪些「無謂的忙碌」來確保「成果」？ |
| **2. 芒格 (決策)**<br>重大投資、失敗復盤、識破邏輯陷阱 | 利用「逆向思維」與跨學科模型識別認知偏差，排除盲點。 | 我準備做 [決策]，請利用「逆向思維」。如果決策最後慘敗，原因是什麼？請結合「激勵機制」幫我排除盲點。 |
| **3. 馬斯克 (創新)**<br>行業壁壘、成本過高、流程冗餘 | 剝離慣例，將系統拆解到物理底層，重新構建最簡路徑。 | 目前項目流程需 [X天]，成本 [Y]。請用第一性原理告訴我：在不違背物理定律下，最快、最省錢的路徑是什麼？ |
| **4. 賈伯斯 (產品)**<br>方案潤色、品牌行銷、追求極致體驗 | 洞察人類對「美」與「簡單」的渴望，刪除平庸，直達本質。 | 這是我的原型。請以最挑剔眼光審視：哪裡是垃圾？哪裡太複雜了？如果只能留一個亮點，該是什麼？幫我刪掉平庸部分。 |
| **5. 原研哉 (審美)**<br>品牌升級、高端視覺、化繁為簡 | 透過「留白」與「減法」重塑事物最本質狀態，讓信息具備「容器感」。 | 方案充斥裝飾，請以原研哉視角幫我找到「本質」。如何透過「留白」和「減法」讓方案高級且充滿可能性？ |
| **6. 李笑來 (進化)**<br>學習阻力、財富困惑、瑣碎人際消耗 | 提升認知維度，將「注意力」視為最寶貴財富，執行長期正確的事。 | 我在 [技能] 進步緩慢。分析一下：我是不是在用勤奮掩蓋思維懶惰？我該如何分配注意力？給我一個符合「定投思維」的方案。 |
| **7. 納瓦爾 (財富)**<br>職業方向、厭惡體力勞動、追求財務自由 | 尋找不隨時間貶值的「專長」，利用代碼、媒體或資本實現槓杆化。 | 我目前工作是 [內容]，感覺在計件賣時間。請分析我哪項技能最難被替代？我該如何加上「代碼」或「媒體」槓杆實現非線性回報？ |

# 摘要：神秘學 — 西洋占星核心理論體系

## 核心摘要
這份摘要涵蓋兩本重量級占星專書的全文本提取：《當代占星研究》(蘇·湯普金斯, 535 頁, 373KB) 和《內在的宇宙》(史蒂芬·阿若優, 191 頁, 168KB)。說真的，這兩本書加起來就是**西洋占星的完整理論框架**——從四元素、星座、行星、宮位到相位，再到解盤方法論。老實講，這套體系的結構化程度完全不輸任何現代知識圖譜。

## 關鍵概念

### 1. 占星學的哲學背景 (《當代占星研究》第一章)
- **宿命論 vs 自由意志**：占星不是決定論，而是「傾向性模型」
- **大宇宙與小宇宙**：As above, so below — 天體運行與人類經驗的對應關係
- **赫爾墨斯秘密教誨**：占星學的古代神秘學根源

### 2. 元素與模式 (第二章)
- **四元素**：火 (行動/意志)、土 (物質/穩定)、風 (智力/溝通)、水 (情感/直覺)
- **元素組合**：星盤中元素的分布揭示個人能量傾向
- **缺乏元素**：某個元素缺失代表該維度的發展課題
- **星座模式**：基本星座 ( initiating )、固定星座 ( sustaining )、變動星座 ( adapting )

### 3. 解盤方法論 (《內在的宇宙》)
- **本質占星學 (Essential Astrology)**：行星、星座、宮位的基本配置
- **演化占星學 (Evolutionary Astrology)**：靈魂進化的視角，北交點/南交點的業力分析
- **心理占星學 (Psychological Astrology)**：榮格心理學與占星的整合
- **解盤指南 (Guidelines)**：系統化的解盤步驟，避免「牌義式」的機械解讀

## 效率提示詞對照表

| 場景 | 效果 | Prompt 示例 |
| :--- | :--- | :--- |
| **文案潤色** | 注入真人溫度與節奏感，消除機械痕跡。 | 「請作為資深編輯，將以下段落進行潤色。要求：刪除冗餘過渡詞，多用動詞，增加感官細節，保持段落簡短（< 3 行），語氣隨性。」 |
| **結構化思維** | 混亂資訊框架化，提取核心行動維度。 | 「分析以下內容並轉化為結構化思維導圖框架。要求：提取 3-5 個核心維度，包含具體執行動作，以 Markdown 表格呈現。」 |
| **創意爆發** | 打破思維定式，提供反直覺選題靈感。 | 「針對 [主題]，提供 10 個不尋常的寫作角度。要求：避開陳詞濫調，從反直覺、跨界或微觀細節出發，配上吸引人的標題。」 |
| **專家模式** | 模擬特定職業口吻，解析底層邏輯與風險。 | 「你現在是擁有 10 年經驗的 [職業]。請針對以下問題提供建議。要求：解釋底層邏輯，並列出 3 個潛在風險點。」 |
| **KM 紀錄自動化** | 快速提取長文精華，產出標準化 KM 格式。 | 「閱讀以下文章並提取核心知識點。要求：產出包含『執行大綱』與『詳細內容』的 Markdown 文件，並加上今日時間戳。」 |

# 實體：[[LLM_Wiki|LLM Wiki]] Dashboard (專案)

## 🧩 核心定義
[[LLM_Wiki|LLM Wiki]] Dashboard 是一個將本地 Markdown 知識庫（基於 Karpathy 邏輯）轉化為視覺化、互動式網頁應用的專案。它具備毛玻璃 UI 質感與雙括號 *Link* 解析能力。

## ⚙️ 關鍵事實
- **技術棧**：React 19, Vite, Node.js Express, TypeScript。
- **核心功能**：樹狀導航、Markdown 渲染、雙括號知識跳躍。
- **倉庫位址**：`https://github.com/skytiger6724/llm-wiki-dashboard`。

# 財務管家技能啟動紀錄

- **時間戳記**: 2026-03-24 14:15
- **執行狀態**: 已啟動 (Activated)
- **類別**: 財務分析 / 技能模組

## 執行大綱
啟動 `finance-tracker` 專家技能，配置具備財務控管、預算編列、現金流優化與投資分析能力的 AI 角色。

## 詳細內容

### 1. 角色定義
此技能將 Gemini 定位為資深財務分析師與財務總監（Controller），專精於：
- **預算管理**: 具備自動化 SQL 查詢能力，可進行季度差異分析。
- **現金流優化**: 內建 Python 類別 `CashFlowManager`，可進行滾動式現金流預測。
- **投資評估**: 具備 `InvestmentAnalyzer` 框架，支持 NPV、IRR、回收期計算。

### 2. 核心規範
- **財務精準度優先**: 所有計算必須經過驗證。
- **合規性與風險控管**: 決策建議需包含風險評估。
- **結構化回報**: 預設使用標準化的財務績效報告模板。

### 3. 可用工具與交付物
- **預算框架**: SQL 基礎的部門預算摘要。
- **預測系統**: 基於 Python 的現金流預測模型（含季節性因子）。
- **報告模板**: 包含執行摘要、成本結構分析、預算 vs 實際分析。

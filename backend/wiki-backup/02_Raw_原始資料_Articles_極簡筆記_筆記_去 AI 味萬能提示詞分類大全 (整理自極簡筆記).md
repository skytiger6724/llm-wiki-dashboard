這份表格整合了筆記中所有的核心公式，旨在幫助您將 AI 生成的冷冰冰內容轉化為具有「人氣」和「溫度」的高質量文章。
|   |   |   |   |
|---|---|---|---|
|情境分類|應用目標|核心提示詞模板 (Prompt Template)|實操技巧 (Tips)|
|:---|:---|:---|:---|
|**基礎去 AI 味**|降低機械感，增加人味|請重寫以下文本，使其更自然、更像人類書寫：減少過於完美的句式，添加適當的不規則表達與口語；避免機械化的段落結構，保留核心意圖。|把「綜上所述」換成「講真」，多用「對吧」、「哎」等虛詞。|
|**文筆潤色**|增強感染力與表現力|請對以下文本進行潤色：改進用詞，使用更精準、生動的詞彙替代抽象陳述；調整句式使表達流暢，增強語言的韻律感。|用具體動詞替代形容詞，如把「非常努力」換成「熬了3個通宵」。|
|**結構與邏輯**|提升清晰度與閱讀率|請重新組織文本，使其更易理解：簡化複雜句式，將長段落分解為短單元；使用明確小標題，確保每個段落只包含一個中心思想。|每段不超過 3 行，多用小標題和數字列表，符合碎片化閱讀習慣。|
|**受眾適配**|調整語氣以契合讀者|請將語氣調整為 [正式/友好/專業/權威等]，適配 [目標受眾]：考慮讀者的知識水平，調整專業術語密度與情感基調。|寫給粉絲就多帶共情，寫給專業人士就保留核心術語。|
|**內容豐富化**|增加細節與真實感|請為文本增加具體細節：添加相關例子、數據或場景；融入個人經歷或軼事，使用主動語態展示與主題的人性化聯繫。|把「很多人用」換成具體的案例「我身邊3個朋友最近都在聊...」。|
|**精簡提煉**|砍掉廢話，直擊重點|請精簡文本，突出核心觀點：識別並保留最關鍵信息點，刪除重複表述與多餘修飾，合併相似觀點，確保每句話都有明確目的。|做減法，留下最有力量的句子，讓觀點像釘子一樣扎入讀者心裡。|
|**節奏與靈動**|優化閱讀節奏與美感|請優化節奏與句式變化：混合使用長短句，交替簡單句與複合句；在適當位置使用問句或感嘆句，創造自然的語言韻律。|偶爾來個極短句如「太絕了！」或「你敢信？」，打破機械重複感。|
|**高級框架**|專業化寫作與仿寫|1. **仿寫**：分析 [樣本] 的風格與語言特徵，並以 [主題] 重新創作。2. **CO-STAR**：結合 Context (背景), Objective (目標), Style (風格), Tone (語調), Audience (受眾) 重新生成內容。|先讓 AI 總結樣本風格達成共識後，再執行具體的仿寫任務。|
|**感官與情感**|注入溫度與畫面感|增加視覺、聽覺、嗅覺等感官細節增強身臨其境感；注入符合語境的情緒傾向（如溫暖治愈/熱血激昂），讓文字像有感而發。|描述細節要能「成像」，讓讀者在腦海中看到具體的畫面。|

# 實體：雲端智能部 (Cloud Intelligence Dept)

## 核心定義
[[Dynasafe]] 內部負責技術轉型、AI 應用落地、雲地混合架構規劃與統一標籤治理的核心部門。其使命是將組織從傳統基礎設施維運商轉化為「智能驅動型技術服務商」。

## 關鍵事實
- **核心架構**：下轄 CS (服務)、AST (技術)、STG (儲存) 三大技術集群。
- **2026 戰略重心**：
    - **HCI & Cloud**：推動超融合基礎設施與公有雲對齊。
    - **AIOps**：導入 AI 輔助與自動化維運計畫。
    - **身分治理**：落實「統一標籤治理」技術架構。
- **人才短板**：自動化與 API 整合、公有雲解決方案 (AWS/GCP/Azure) 達標率低。

## 映射至 CIOS 架構
- **感知層 (Sensing)**：全網標籤識別、流量探測。
- **決策層 (Logic)**：意圖導向策略編排、AI 使用情境判定。
- **行動層 (Action)**：自動化維運腳本執行、SD-WAN 路徑調控。

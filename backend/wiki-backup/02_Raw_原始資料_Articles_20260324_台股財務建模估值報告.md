# 20260324_台股財務建模估值報告 (DCF Model)

**時間戳：** 2026-03-24 17:25

## 執行大綱
老實說，單看 ROI (投資報酬率) 只能看到過去，而財務建模 (Financial Modeling) 則是用來洞察未來。這次任務是針對近期表現最優異的 **欣興 (3037.TW)** 建立一個 **現金流量折現模型 (DCF)**。透過預測未來五年的自由現金流 (FCF) 並進行折現分析，推算出企業的合理內在價值。

## 詳細內容

### 1. 模型核心假設 (Assumptions)
- **初始自由現金流 (Base FCF)：** 模擬假設為 200 億 TWD。
- **預期成長率 (Growth Rate)：** 未來 5 年假設維持 15% 的高成長。
- **折現率 (WACC)：** 基準設定為 10% (包含風險溢價)。
- **永續成長率 (Terminal Growth)：** 設為保守的 2%。

### 2. 估值結果摘要
- **未來五年 FCF 現值總計：** 1144.92 億 TWD。
- **終端價值 (TV) 現值：** 3184.68 億 TWD (佔總估值約 73.5%)。
- **企業內在價值 (EV)：** **4329.6 億 TWD**。

### 3. 敏感度分析 (Sensitivity Analysis)
說真的，估值最怕的就是「算死」，所以我們得看範圍。

| WACC (%) | 內在價值估算 (億 TWD) | 相對基準變動 (%) |
| :--- | :--- | :--- |
| 8.0% | 5866.33 | +35.5% |
| 9.0% | 4987.45 | +15.2% |
| **10.0%** (Base) | **4329.60** | **0.0%** |
| 11.0% | 3819.03 | -11.8% |
| 12.0% | 3411.53 | -21.2% |

### 4. 專業財務觀察
說真的，終端價值佔了總估值的 73%，這代表 **3037.TW** 的長期競爭力 (如製程技術、市佔率) 比短期的現金流波動更為重要。如果未來營收成長率能上修到 20%，估值將會有更顯著的跳升空間。

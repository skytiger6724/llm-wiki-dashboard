# 核心概念：西方占星體系 (Western Astrology System)

## 核心定義
西方占星體系是一種基於黃道十二星座、十大行星、十二宮位與相位角度的象徵推理系統。它透過觀測天體運行與地球生命事件之間的「共時性 (Synchronicity)」，來解構人格、預測趨勢並理解生命意圖。

## 體系架構 (CIOS 映射)
- **感知層 (Sensing)**：天文觀測與星曆數據（星盤起盤）。
- **語義層 (Semantic)**：行星、星座、宮位的符號定義。
- **決策層 (Logic)**：相位互動與廟旺落陷的權重判定。
- **行動層 (Action)**：解盤建議與流年應對策略。

- **心理學映射**：行星即是 [[榮格原型理論_Jungian_Archetypes|人格原型]]。
- **硬件載體**：行星能量可透過特定 [[晶體能量與頻率共振_Crystal_Resonance|水晶頻率]] 進行調頻。
- **個人行星**：太陽、月亮、水星、金星、火星。
- **社會行星**：木星、土星。
- **世代行星**：天王星、海王星、冥王星。

### 2. 能量特質：黃道十二星座
- **[[四大元素_Four_Elements|四大元素]]**：火 (Fire)、土 (Earth)、風 (Air)、水 (Water)。
- **三種模式**：基本 (Cardinal)、固定 (Fixed)、變動 (Mutable)。

### 3. 生命領域：[[宮位系統_House_System]]
- 描述能量在哪些具體生活場景（如財富、事業、感情）中顯現。

### 4. 能量互動：[[相位系統_Aspect_System]]
- 行星間的幾何角度（0°, 60°, 90°, 120°, 180°）決定了能量是流暢還是衝突。

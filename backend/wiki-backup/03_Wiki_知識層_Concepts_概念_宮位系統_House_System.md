# 概念：宮位系統 (House_System)

## 核心定義
宮位系統將星盤分為十二個扇形區域，代表個人生命中不同的「生活領域」。如果行星是能量（What），星座是表現方式（How），那麼宮位就是能量顯現的具體場景（Where）。

## 十二宮位解析

| 宮位 | 名稱 | 關鍵詞 | 生命領域 |
|:---|:---|:---|:---|
| **第一宮** | 命宮 (ASC) | 自我、形象 | 個人形象、第一印象、體質、人格面具。 |
| **第二宮** | 財帛宮 | 價值、金錢 | 財務狀況、動產、自我價值感、賺錢能力。 |
| **第三宮** | 兄弟宮 | 溝通、學習 | 早期教育、兄弟姊妹、短途旅行、心智運作。 |
| **第四宮** | 田宅宮 (IC) | 根源、家庭 | 晚年生活、父親/母親、不動產、內在安全感。 |
| **第五宮** | 子女宮 | 創造、戀愛 | 創意表現、戀愛、子女、投機、遊戲、快樂。 |
| **第六宮** | 奴僕宮 | 工作、健康 | 日常勞務、健康狀況、部屬、寵物、生活紀律。 |
| **第七宮** | 夫妻宮 (DES) | 關係、合作 | 婚姻伴侶、合夥人、公開的敵人、法律契約。 |
| **第八宮** | 疾厄宮 | 轉化、共有 | 深層情感、性、他人的錢財、遺產、死亡、神秘學。 |
| **第九宮** | 遷移宮 | 高等、遠行 | 哲學、宗教、高等教育、長途旅行、國外事務。 |
| **第十宮** | 官祿宮 (MC) | 事業、地位 | 社會地位、職業成就、名聲、母親/父親。 |
| **第十一宮** | 福德宮 | 社群、願景 | 朋友、團體組織、人道主義、長遠目標與希望。 |
| **第十二宮** | 玄秘宮 | 潛意識、隱密 | 靈性探索、隱藏的敵人、醫院、監獄、前世業力。 |

## 關鍵事實
- **四軸點 (The Four Angles)**：
    - **ASC (上升)**：第一宮起點，自我覺醒。
    - **IC (天底)**：第四宮起點，靈魂根源。
    - **DES (下降)**：第七宮起點，他人關係。
    - **MC (中天)**：第十宮起點，社會頂峰。
- **宮位分類**：
    - **始宮 (Angular)**：1, 4, 7, 10（開創能量）。
    - **續宮 (Succedent)**：2, 5, 8, 11（穩定能量）。
    - **終宮 (Cadent)**：3, 6, 9, 12（變動能量）。

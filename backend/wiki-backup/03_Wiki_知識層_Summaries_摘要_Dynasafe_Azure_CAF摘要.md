## 核心摘要

Microsoft Cloud Adoption Framework (CAF) for Azure 是一套完整的生命週期架構，協助企業有策略地採用雲端服務。框架涵蓋八大方法：策略、計畫、就緒、移轉、創新、安全、管理、控管。企業採用 CAF 能降低雲端轉型風險，確保技術投資與業務目標一致。Governance Benchmark 工具可評估當前治理狀態與目標差距。Enterprise-scale Landing Zone 提供參考架構，包含身分管理、網路連接、治理策略等設計區域。

## 關鍵概念清單

- **雲端採用生命週期八大理面**：策略(Strategy)、計畫(Plan)、就緒(Ready)、移轉(Migrate)、創新(Innovate)、安全(Secure)、管理(Manage)、控管(Govern)
- **Landing Zone 設計區域**：身分與存取管理、網路拓撲、治理策略、安全管理、監控與營運
- **管理群組階層**：Tenant Root Group → Platform → Landing Zones → Decommissioned / Sandbox
- **Governance Benchmark 工具**：評估當前治理狀態、設定商業優先順序、識別改進差距
- **Hub-Spoke 與 VWAN 架構**：Azure 登陸區域的兩種主流網路拓撲模式
- **策略性移轉評量(SMART)**：評估雲端移轉整備程度的評估工具
- **CISO Workshop**：零信任架構、現代化安全營運、身分保護、基礎設施安全、資料安全與 GRC、IoT/OT 安全六大模組
- **CloudLabs Immersion Workshop**：Hands-on 實驗室環境，涵蓋 .NET 現代化、AI、Analytics、Cloud Native Apps、Data Modernization、Infrastructure Migration 等軌道
- **FastTrack for Azure**：Microsoft 工程師提供的免費技術支援服務

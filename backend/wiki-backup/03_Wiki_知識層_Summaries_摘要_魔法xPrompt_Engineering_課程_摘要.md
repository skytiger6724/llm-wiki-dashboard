# 摘要：魔法 × Prompt Engineering 實戰課程 v1.0

## 核心摘要
結合 LLM Wiki 知識庫中的 Prompt Engineering 知識體系，設計一門面向已有 Prompt 經驗者的實戰進階課程。四大模組（魔法覺醒/儀式魔法/鍊金術/魔法師之路）共 12 小時，核心主張：「在專家層級，提示詞工程本質上是認知架構。AI 只是執行引擎。」包含完整課程大綱和 ~120 頁 PPT 授課框架。

## 關鍵概念
- **Module 1 魔法覺醒**：LLM 心智模型、Context Engineering（context.json）、隧道效應
- **Module 2 儀式魔法**：RTCO 框架、CO-STAR 框架、角色/約束/輸出格式設計
- **Module 3 鍊金術**：CoT/Few-shot/Chaining/Multi-Agent/Meta-Prompting
- **Module 4 魔法師之路**：Prompt 庫管理、業界 System Prompt 拆解、畢業專案
- **教學方法論**：概念引入 → 原理拆解 → 現場實操 → 獨立練習 → 總結
- **Wiki 來源**：7 份 Wiki 知識文件（如何掌握 Prompt Engineering、提示詞模板整理、prompts.csv 等）

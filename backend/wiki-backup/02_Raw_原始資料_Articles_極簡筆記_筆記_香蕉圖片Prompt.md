# **小編對於AI生圖感受和心得**
從上，只是小編部分整合了2025年生圖的prompt，有些更是自己玩的，隨意分享。我認為AI發展得很快，同時技術日漸成熟。
記得最初的AI生圖情況是: 即使我下了清晰指令，而AI亦_**不能**_精準處理我的要求。與現在相比，簡直是**大相逕庭**！
所以，我們除了學習使用AI技巧之餘，同時要增加自已對於不同領域的認知，例如 : 生成圖片、生成影片、學會詳細並精準表達自己想法 等等……，當這些軟性技能裝備好自己後，便可以精準下達指令，更活用AI！
我認為AI不可以完全取代人類，而在未來，我們亦要好好裝備自己 (例如:閱讀、培養興趣…)，增廣見聞！才可好好使用AI工具幫助自己！

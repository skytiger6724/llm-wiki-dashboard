### 2. 四元素能量互動理論
- **同元素叠加** → 能量加強 (火+火=更活躍, 水+水=更情緒化)
- **同屬性叠加** → 互相增強 (火+風=高活躍不穩定, 水+土=深沉穩定)
- **異屬性叠加** → 互相削弱 (火+土=受阻滯, 風+水=理性vs感性拉扯)
- **元素不足** → 該維度的能力缺失 (缺風=缺乏溝通, 缺火=缺乏目標)
- **元素過度** → 負面特質顯現 (水過度=情緒化依賴, 火過度=暴躁衝動)

### 3. 實戰案例解牌方法論
從 15+ 個案例中提取的標準解牌流程：

**Step 1：牌面解讀**
- 每張牌的正/逆位含義
- 元素屬性分析
- 在牌陣位置的時間維度 (過去/現在/未來)

**Step 2：元素場分析**
- 哪種元素過度/不足
- 元素組合的互動關係
- 缺失元素代表的「盲點」

**Step 3：因果鏈推理**
- 過去牌 → 現在牌 → 未來牌的因果關聯
- 隱藏的「問題根源」往往在過去牌中
- 未來牌不僅是結果，更是「需要注意的隱患」

**Step 4：建議生成**
- 根據缺失元素給出具體行動建議
- 例：缺風 (溝通) → 建議「深度對話」
- 例：缺火 (目標) → 建議「設定共同目標」

### 4. 典型案例模式
| 模式 | 牌陣特徵 | 解讀要點 |
|:---|:---|:---|
| **一見鍾情型** | 火元素開頭 → 土元素穩定 → 未來隱患 | 快速發展導致缺乏了解 |
| **得過且過型** | 缺火元素 + 停滯牌 (星幣四) | 習慣性維持，缺乏推進力 |
| **慾望主導型** | 惡魔牌 + 聖杯三 | 不純粹的感情，易有多方關係 |
| **破局翻轉型** | 逆位問題牌 → 正位行動牌 → 穩定牌 | 積極行動可化解過去隱患 |

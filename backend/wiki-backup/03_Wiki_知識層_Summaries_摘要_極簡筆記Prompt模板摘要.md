# 摘要：極簡筆記Prompt模板

## 核心摘要
極簡筆記包含30+個Prompt模板文件，涵蓋6個高槓桿提示詞(深度研究員、內容變異引擎)、10個NotebookLM高級提示詞、5個AI高階提示詞框架、精准指令提问模板、AI去味萬能提示詞分類大全，以及圖像生成提示詞(人物肖像、明星合照、香蕉修圖等)。核心洞察：2026年真正拉開差距的是把AI當深度研究員、內容變異引擎和首席戰略顧問。

## 關鍵概念
- [[極簡筆記Prompt模板摘要|High-Leverage Prompts]]: 6個商業實戰提示詞(深度研究、內容變異、數據驅動、歷史映射等)
- *內容變異引擎*: 1個舊想法裂變為10條高價值內容形態
- [[極簡筆記Prompt模板摘要|Precise Prompt Templates]]: 可直接複製使用的Prompt模板，收藏級
- [[極簡筆記Prompt模板摘要|AI De-AI Prompts]]: 多樣用詞、感官細節、真實情感、CO-STAR框架

## 關鍵實體
- *NotebookLM提示詞*: 10個高級提示詞將NotebookLM變為高效生產力工具
- *Awesome Prompt*: Prompt集合，包含ChatGPT Prompts Library

## 文件清單 (部分)
- 6 個直接產生商業結果的高槓桿提示詞 (151行)
- 10個NotebookLM高級提示詞直接複製到NotebookLM使用
- 五個AI高階提示詞(Prompt)框架
- 精准指令提问prompt模板，可直接複製使用
- 去 AI 味萬能提示詞分類大全 (整理自極簡筆記)
- 商務AI prompt
- 人物提示詞、專業人像照 Prompt
- 明星合照大片Prompt、AI合照
- 香蕉修圖Prompt系列 (3個版本)
- 照片生成Prompt、照片提示詞

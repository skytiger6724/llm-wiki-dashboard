# 極簡筆記：提示詞模板整理表格

本表格整理自 Notion 中的各類提示詞筆記，涵蓋通用框架、職場溝通、學術研究、視覺製圖、新媒體運營及內容潤色等場景。

| 使用場景 | 應用 | 主要功能 | 提示詞模板 |
| :--- | :--- | :--- | :--- |
| **通用框架** | **RTCO 框架** | 全場景通用指令優化 | **Role**: 你是一位[身分/專家]。<br>**Task**: [具體任務描述]。<br>**Constraint**: [約束條件，如字數、禁忌詞、語氣]。<br>**Output**: [輸出格式要求]。 |
| **通用框架** | **5W1H 框架** | 資訊完整性提煉 | 請幫我生成一份工作任務簡報，需包含：<br>**Who**: [執行者]；**What**: [任務內容]；**When**: [截止時間]；**Where**: [發布平台]；**Why**: [目的]；**How**: [具體要求/風格]。 |
| **通用框架** | **CO-STAR 框架** | 專業化內容生成 | 結合 **Context** (背景), **Objective** (目標), **Style** (風格), **Tone** (語調), **Audience** (受眾) 重新生成內容。 |
| **職場溝通** | **PREC 框架** | 職場說理與申請 | 請用 PREC 框架組織一段向[對象]申請[事項]的陳述：<br>**Point**(觀點)、**Reason**(理由)、**Example**(案例)、**Conclusion**(結論)。 |
| **職場溝通** | **會議紀要** | 快速整理會議重點 | 整理【會議主題】紀要，包含：會議時間、人員、核心議題、關鍵決策、待辦事項（明確責任人與時間）、未解決問題。 |
| **職場溝通** | **個人年終總結** | 量化成果與規劃 | 撰寫年終總結，包含：1. 核心工作成果（量化數據）；2. 工作不足（客觀真實）；3. 下一年度規劃。字數 1000 以內。 |
| **行銷創意** | **AIDA 模型** | 撰寫行銷推廣文案 | 請以 AIDA 模型撰寫[產品名稱]的行銷文案：<br>**Attention**: [吸引注意的一句話]；**Interest**: [產品核心原理/興趣點]；**Desire**: [用戶好處/場景描繪]；**Action**: [明確行動號召]。 |
| **行銷創意** | **SCAMPER 框架** | 產品或想法創意改進 | 請對[產品/想法]進行創意改進，針對以下維度提出創新點：<br>**Substitute**(替代)、**Combine**(合併)、**Adapt**(調整)、**Modify**(修改)、**Put to other uses**(他用)、**Eliminate**(消除)、**Reverse**(重組)。 |
| **學術研究** | **開題與框架搭建** | 論文結構與創新分析 | 基於主題《[具體主題]》，結合該領域近 3 年頂刊研究熱點，拆解核心與次要研究問題，搭建完整論文框架，標註核心寫作要點與創新點。 |
| **學術研究** | **學術潤色與降重** | 提升嚴謹性與符合規範 | 潤色以下論文段落 [貼上內容]，適配 [目標場景] 語言風格，修正語法錯誤，強化邏輯連接，提升論證嚴謹性。 |
| **視覺製圖** | **專業/超寫實肖像** | 生成高品質人物影像 | Create a hyper-realistic portrait of [Subject] with [Features] in [Attire]. Use [Lighting, e.g., low-key diffused], [Background]. High detail, natural skin tones, cinematic color grading. |
| **視覺製圖** | **爆款風格轉換** | 社交媒體流行視覺 | (以吉卜力為例) Turn this photo into a Ghibli-style animated character. Use soft watercolor tones, expressive eyes, and rich nature background. |
| **視覺製圖** | **客製化玩具公仔** | 趣味品牌/個人展示 | Create a stylized action figure of myself in a plastic blister pack, designed like a premium collectible toy. Include header "[Name]" and subheadings. |
| **內容潤色** | **基礎去 AI 味** | 降低機械感，增加人味 | 請重寫以下文本，使其更自然：減少完美句式，添加適當的不規則表達與口語；把「綜上所述」換成「講真」，多用虛詞。 |
| **內容潤色** | **感官與情感注入** | 增加畫面感與溫度 | 增加視覺、聽覺、嗅覺等感官細節；注入符合語境的情緒（如溫暖治愈/熱血激昂），讓文字像有感而發。 |
| **內容潤色** | **精簡提煉** | 砍掉廢話，直擊重點 | 請精簡文本，識別並保留最關鍵資訊點，刪除重複表述與多餘修飾，確保每句話都有明確目的。 |
| **技術開發** | **安全代碼審查** | 排查漏洞與敏感資訊 | 請對以下 [Code] 進行深度安全分析。重點：1. 漏洞排查 (SQLi, XSS)；2. 敏感資訊洩漏；3. 認證與授權邏輯。 |
| **產品管理** | **PRD 自動生成** | 需求轉化為標準文檔 | 根據 [需求描述] 生成 PRD，包含：1. 產品概述；2. 使用場景；3. 功能細節說明；4. 業務流程；5. 驗收標準。 |
| **行政/HR** | **兩階段簡歷篩選** | 技能匹配與深度分析 | 1. **初篩**: 匹配關鍵技能與學歷；2. **深挖**: 分析專案成就，輸出勝任力報告及建議面試問題。 |
